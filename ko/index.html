우리에 대하여

우리는 프로페셔널 웹 개발 과정 중 하나입니다

웹 제작에 대한 이해를 제공하며, 초보자를 위한 기초 프로그래밍 언어 수업을 포함합니다

  • HTML 및 CSS 주제 소개.
  • HTML 태그를 사용하여 조건을 이해하고 만들기.
  • CSS로 레이아웃 위치 이해하기.
  • 웹 구축을 위한 HTML 및 CSS 통합.
  • 웹에서 애니메이션을 만들기 위한 JavaScript 이해.
  • React Js의 이해 및 웹 페이지에 사용하기.

우리의 메뉴

웹사이트를 만드는 데 필요한 다양한 자료를 제공합니다. 초보부터 고급 수준까지 다양한 레벨의 자료를 제공합니다.

HTML

HTML(Hypertext Markup Language)은 웹 페이지와 웹 애플리케이션을 만들기 위해 사용되는 표준 마크업 언어입니다.


더 읽기

CSS

CSS(Cascading Style Sheet)는 HTML과 같은 마크업 언어로 작성된 요소의 디자인을 꾸미는 데 사용됩니다.


더 읽기

JavaScript

JavaScript는 웹 개발자가 웹 페이지에서 더 동적인 상호 작용을 만들기 위해 사용하는 스크립트 언어입니다.


더 읽기

React Js

ReactJS 또는 React는 페이스북에서 개발된 매우 인기 있는 JavaScript 라이브러리로, 모바일 및 웹 애플리케이션 개발에 사용됩니다.
